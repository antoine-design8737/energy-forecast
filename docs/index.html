Under the hood

How the forecast is built

Three years of real grid data, physically-motivated features, and a leakage-free test.

🛰️

Real grid data

Hourly load and generation-by-source from the ENTSO-E Transparency Platform, merged with population-weighted weather from Open-Meteo onto one clean UTC grid.

— · France
🧮

Engineered features

Cyclically-encoded calendar terms, public holidays, load lags & rolling stats, plus temperature (with a squared term for the U-shaped demand curve), wind and solar radiation.

— predictive features
🎯

Honest evaluation

Walk-forward validation: the model is retrained on all prior data before each weekly test block, then scored on it. This mirrors real deployment and rules out look-ahead bias.

No leakage · 90-day hold-out
